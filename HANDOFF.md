- **`standalone/*.html`** — полностью инлайн-версии (React, рантайм, шрифты и картинки
  вшиты). Открываются двойным кликом в любом браузере и выглядят **точно как в макете**,
  даже без интернета. Это визуальный эталон.
- **`source/*.dc.html`** — исходный код прототипа. Вся вёрстка здесь.

## 2. Про формат исходных файлов (важно)

Файлы `.dc.html` — это **дизайн-референс, собранный в HTML-прототипе**, а не продакшн-код
для копирования один-в-один. Технически это компонент, который рантайм `support.js`
рендерит через React. Структура каждого файла:

- **Шаблон** между `<x-dc>…</x-dc>` — разметка со **всеми стилями инлайн** (`style="…"`).
  Отдельных CSS-файлов нет — это осознанно, чтобы всё красилось сразу.
- **`<script data-dc-script>`** внизу — класс `Component extends DCLogic`. В нём:
  - данные (тексты пунктов, масок, услуг, глав книги) в `renderVals()`;
  - **все фирменные SVG-знаки** (логотип-древо, круги целостности, линии дыхания,
    ghost-древо в герое, интерактивное «Древо жизни») рисуются через
    `React.createElement('svg', …)` — ищите методы `logo()`, `circles()`, `heroMark()`,
    `tree()`, `sprig()`, `breathMark()` и т.д.
  - плейсхолдеры `{{ … }}` в шаблоне подставляются из `renderVals()`.
- Адаптив — через `@media` в `<helmet><style>` (только там, где инлайн невозможно) и через
  `srcset`/`<source media>` в `<picture>`.

**Задача при переносе:** воссоздать эти экраны в целевом стеке. По исходному брифу это
**Astro, статика, mobile-first, self-hosted шрифты**. То есть: разложить инлайн-стили в
компоненты Astro/CSS, вынести SVG в `.svg`-файлы или Astro-компоненты, подключить шрифты
локально. Ниже — все токены и параметры, чтобы повторить пиксель-в-пиксель.

### Цвета (палитра строгая — только эти)
| Роль | HEX |

| Светло-бежевый (фон базовый) | `#EADFCF` |
| Тёплый золотой (акцент-точки/линии/номера, НЕ заливка) | `#EBC37C` |
| Графитовый (тёмный фон, глубина) | `#36413E` |
| Зелёный тёмный (фон секции книги/финала) | `#2E5A4C` |
| Обложка книги (мокап, тёмный) | `#2B332F` |

Вспомогательные оттенки текста (производные, для абзацев): `#3D5A4F`, `#4A6459`,
`#C9C1AC` (светлый на тёмном), `#D0C8B4` (светлый на тёмном).

**Правила:** фоны секций — только `#EADFCF` / `#36413E` / `#2E5A4C`. Белый как фон
секции не используется (белым бывают только карточки на бежевом). Золотой `#EBC37C` —
только акцентом (точки, тонкие линии, номера), никогда заливкой. Кнопки — только
терракота. Красных CTA, градиентов вне палитры, эмодзи-иконок нет.

### Типографика (сейчас подключено с Google Fonts CDN → при переносе self-host woff2)
- **Cormorant Garamond** — заголовки, цитаты, названия. Начертания 400/500/600 + italic.
- **Lora** — основной текст (абзацы). 400/500 + italic.
- **Roboto** — интерфейс, подписи, кнопки, надзаголовки, дескрипторы. 300/400/500.

Ключевые размеры:
- H1 героя: `clamp(46px, 6.1vw, 78px)`, Cormorant 500, line-height 1.04.
- H2 секций: `clamp(30px, 3.6vw, 46px)`, Cormorant 500, line-height 1.1.
- Надзаголовок (kicker): Roboto, 12.5px, `letter-spacing:2.5px`, uppercase, `#B36A4E`
  (на тёмном — `#EBC37C`).
- Абзац: Lora 17–18px, line-height 1.6–1.7.
- Кнопка/бейдж: Roboto 15px.

### Радиусы, тени, отступы
- Радиусы: кнопки/бейджи `44px` (pill); карточки `12–14px`; фото героя слева `16px 0 0 16px`.
- Тень карточек (hover): `0 18px 40px -24px rgba(54,65,62,.4)`.
- Тень фото героя: `-34px 26px 100px -55px rgba(54,65,62,.55)` (мягкая, широкая).
- Контейнер: `max-width:1180px; margin:0 auto; padding:0 32px`.
- Вертикальные отступы секций: `~84px` сверху/снизу (`padding: 84px 32px`).
- Разделители: `1px solid rgba(46,90,76,.12)` на светлом; `rgba(234,223,207,.14)` на тёмном.

## 6. Интерактив и поведение
- **Древо жизни (главная):** узлы кликабельны и реагируют на hover; активный узел
  подсвечивается золотым пульсирующим кольцом (`@keyframes breathe`). Полосы-переключатели
  под карточкой дублируют выбор. Прорастание — один раз при скролле в вьюпорт.
- **FAQ и Оглавление:** аккордеоны, одно открыто за раз, анимация `max-height .4–.5s ease`.
  Иконка `+`/`–`.
- **Карточки (маски, услуги, отзывы):** hover — лёгкий подъём + тень.
- **Кнопки:** hover меняет заливку терракоты на `#9C5A41` (у контурных — усиление рамки).
- **Ссылки CTA:** сейчас заглушки `#whatsapp` / `#telegram` / `#vk` (в коде это пропсы
  `waLink` / `tgLink` / `vkLink`). Подставить реальные ссылки на мессенджеры.
- **Плавный скролл** по якорям меню (`scroll-behavior:smooth`).

отдельные `.svg` / компоненты — геометрия и цвета уже готовы, копируйте из методов класса.
